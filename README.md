## 3. Tokenomics and Economic Model

The Nexus is built on a sustainable, **deflationary** economic model using the native **$QUBIC** token. $QUBIC is the sole medium for Entry Fees/Buy-ins, Rewards/Payouts, and Passive Income Distribution.

### Revenue Distribution Model (Standardized across all titles)

A standardized distribution mechanism is applied to the fee taken from every tournament pot:

| Allocation | Percentage | Purpose |
| :--- | :--- | :--- |
| **Player Prize Pool** | **75%** | Distributed to top-scoring/winning players. |
| **IPO Shareholders** | **20%** | Automatically paid to holders of the Nexus IPO share token as passive yield. |
| **Dev Team / Treasury** | **3%** | Funding for development, maintenance, and ongoing ecosystem growth. |
| **Deflationary Burn / Liquidity** | **2%** | Permanently removes $QUBIC from circulation, or adds to a community-chosen liquidity pool (*PaQman* only). |

This mandatory 2% burn introduces persistent deflationary pressure on the overall **$QUBIC** supply.

## 6. Security and Fair Play

Trust is paramount. The Nexus ensures this through:
* **Provable Fairness:** All RNG is verified by decentralized Qubic quorums. Arcade session results are cryptographically hashed and validated on-chain.
* **Anti-Cheat Measures:** Deterministic game constraints (*snaQe*) and commit-reveal logic for card shuffling (*poQer*/*Qbust*).
* **Immutable Logic:** All financial logic is handled by audited Qubic smart contracts and cannot be changed post-deployment.
* **Non-Custodial:** Player funds are held in escrow only during tournament duration, guaranteeing instant and automatic distribution.
